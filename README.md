# VPS Initialize

Ubuntu VPS initialization script.

## Quick Start

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/naico-wang/vps-initialize/main/install.sh)
```

## Full Install

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/naico-wang/vps-initialize/main/install.sh) all
```

## Features

- Ubuntu upgrade
- UFW firewall
- Nginx
- acme.sh
- Wildcard SSL
- 3x-ui
- Zsh
- Starship
- eza
